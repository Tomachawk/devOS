Projekt: devOS

Opis:
Budujemy futurystyczny interfejs webowy w stylu sci-fi HUD (jak Jarvis / systemy wojskowe / cyberpunk dashboard).

Styl:
- ciemne tło (#0b0f14)
- neonowy cyan (#00f0ff)
- akcenty pomarańczowe (#ff7a00)
- świecące elementy (glow)
- cienkie linie, siatki, kontury
- niestandardowe ramki (nie zwykłe rounded-xl)
- efekt "system operacyjny", nie SaaS

Technologie:
- React
- Tailwind CSS

Zasady:
- nie tworzymy generycznych dashboardów biznesowych
- każdy komponent musi pasować do stylu devOS
- komponenty mają być modularne i gotowe do rozbudowy
